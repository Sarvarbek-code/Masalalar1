Masala: "Aqlli uy xavfsizlik tizimi optimizatori"
Foydalanuvchi aqlli uy xavfsizlik tizimini boshqarishni xohlaydi va quyidagi ma‘lumotlarni kiritadi:
- Xavf darajasi (xavf, butun son, 0-100, xavfsizlik holatini ko‘rsatadi).
- Harakat sensori (harakat, Boolean: True yoki False, uyda harakat aniqlanganmi).
- Eshik sensori (eshik, Boolean: True yoki False, eshik ochilganmi).
- Vaqt (soat, butun son, 0-23, 24 soat formatida).
- Energiya darajasi (energiya, %, haqiqiy son, 0-100, tizimning batareya zaryadi).
- Internet aloqasi (internet, Boolean: True yoki False).
- Foydalanuvchi rejimi (rejim, matn: "uyda", "tashqarida", "ta’til").

Vazifa: Xavfsizlik tizimi qanday harakat qilishi kerakligini aniqlang (signalizatsiya, ogohlantirish, monitoring yoki hech narsa) va tizimning energiya sarfini, shuningdek, yangi batareya darajasini hisoblang. 

Quyidagi shartlarni hisobga oling:

1. Xavfsizlik harakatlari va energiya sarfi:
   - Signalizatsiya:
     - Faqat xavf 80 dan yuqori bo‘lsa yoki (harakat = True va eshik = True) faollashadi.
     - Energiya sarfi: 10% batareya/saat.
     - Internet bo‘lsa, politsiyaga xabar yuboriladi; bo‘lmasa, faqat ovozli signal.
     - Narxi: 50,000 so‘m (politsiyaga xabar uchun) yoki 10,000 so‘m (faqat ovozli signal).
   - Ogohlantirish:
     - xavf 50-80 oralig‘ida bo‘lsa yoki (harakat = True va eshik = False) faollashadi.
     - Energiya sarfi: 5% batareya/saat.
     - Internet bo‘lsa, telefon orqali xabar; bo‘lmasa, LED yonadi.
     - Narxi: 5,000 so‘m (telefon xabari) yoki 1,000 so‘m (LED).
   - Monitoring:
     - xavf 20-50 oralig‘ida bo‘lsa yoki hech qanday sensor faol bo‘lmasa (harakat = False, eshik = False).
     - Energiya sarfi: 2% batareya/saat.
     - Narxi: 500 so‘m/saat.
   - Hech narsa:
     - xavf < 20 va sensorlar faol bo‘lmasa.
     - Energiya sarfi: 0%.

2. Rejim shartlari:
   - "uyda":
     - Signalizatsiya faqat xavf > 90 bo‘lsa ishlaydi.
     - Ogohlantirish xavf 30-90 oralig‘ida bo‘lsa faollashadi.
     - Monitoring doimiy ravishda ishlaydi, agar boshqa harakat shartlari bajarilmasa.
   - "tashqarida":
     - Signalizatsiya xavf > 70 yoki (harakat = True va eshik = True) bo‘lsa faollashadi.
     - Ogohlantirish xavf 40-70 oralig‘ida yoki (harakat = True va eshik = False) bo‘lsa.
     - Monitoring qolgan hollarda.
   - "ta’til":
     - Signalizatsiya xavf > 60 yoki (harakat = True yoki eshik = True) bo‘lsa faollashadi.
     - Ogohlantirish xavf 30-60 oralig‘ida bo‘lsa.
     - Monitoring qolgan hollarda.

3. Vaqtga bog‘liq shartlar:
   - Tunda (22:00-6:00) signalizatsiya yoki ogohlantirish narxi 20% qimmatlashadi.
   - Kunduzi (6:00-22:00) monitoring narxi 10% arzonlashadi.

4. Energiya cheklovlari:
   - Agar energiya < 10% bo‘lsa, signalizatsiya ishlamaydi, ogohlantirish faollashadi.
   - Agar energiya < 5% bo‘lsa, faqat monitoring ishlaydi.
   - Agar energiya = 0%, tizim o‘chadi (“Tizim o‘chirilgan” xabari).

5. Aksiya shartlari:
   - Agar internet = True va rejim = "ta’til", signalizatsiya narxi 15% arzonlashadi.
   - Agar soat 12:00-15:00 bo‘lsa va xavf < 50, ogohlantirish narxi 10% arzonlashadi.
   - Agar rejim = "uyda" va xavf < 30, monitoring bepul.

6. Xato holatlari:
   - Agar xavf 0-100 oralig‘ida bo‘lmasa, “Xato xavf darajasi” chiqariladi.
   - Agar soat 0-23 oralig‘ida bo‘lmasa, “Noto‘g‘ri vaqt” chiqariladi.
   - Agar energiya 0-100 oralig‘ida bo‘lmasa, “Xato energiya darajasi” chiqariladi.
   - Agar rejim belgilangan qiymatlardan biri bo‘lmasa, “Noto‘g‘ri rejim” chiqariladi.

Kiruvchi ma‘lumotlar:
- xavf — xavf darajasi (butun son, 0-100).
- harakat — harakat sensori (Boolean).
- eshik — eshik sensori (Boolean).
- soat — vaqt (butun son, 0-23).
- energiya — batareya zaryadi (%, haqiqiy son, 0-100).
- internet — internet aloqasi (Boolean).
- rejim — foydalanuvchi rejimi (matn: "uyda", "tashqarida", "ta’til").

Chiquvchi ma‘lumot:
- Tizimning harakati (signalizatsiya, ogohlantirish, monitoring yoki hech narsa).
- Umumiy xarajat (so‘m).
- Yangi batareya zaryadi (%).
- Agar xato bo‘lsa, mos xato xabari.
